# Arria 10

 - tcl scripts

Scripts for programming Arria 10 flash.

## Description

 - `cfi1616.tcl`

## DE5a Flash Memory Map

PFL option bits 0x00030000-0x0003FFFF
Factory hardware 0x00040000-0x02B3FFFF
User hardware 0x02B40000-0x0563FFFF
Factory software 0x05640000-0x05E3FFFF
User software and data 0x05E40000-0x0FFFFFFF
