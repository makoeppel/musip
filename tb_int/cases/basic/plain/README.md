# `tb_int/cases/basic/plain`

Plain mixed-language replay bench for the SWB integration.

This is the focused deterministic replay path:

- no UVM classes,
- deterministic replay vectors generated by `../ref/`,
- plain VHDL testbench around the existing narrow DUT wrapper.

Use this when:

1. you want a smaller replay-only bench than the full UVM harness,
2. you want to replay the exact reference case in RTL,
3. you do not need UVM classes for the run you are debugging.

Recommended order:

1. Run `make ip-tlm-basic-smoke`
2. Run `make ip-check-license`
3. Run `make ip-plain-basic-smoke`
4. Run `make ip-tlm-basic`
5. Run `make ip-plain-basic`
6. If you want the split OPQ boundary harness instead, run `make ip-plain-basic-2env`

The local `run` target defaults to the full replay bundle and `USE_MERGE=1`.
The local `run-smoke` target defaults to `../ref/out_smoke` and `USE_MERGE=1`.

Primary local targets:

- `make compile`
- `make run`
- `make run-smoke`
- `make clean`

Helpful overrides:

- `make run REPLAY_DIR=/absolute/path/to/tb_int/cases/basic/ref/out`
- `make run-smoke SMOKE_REPLAY_DIR=/absolute/path/to/tb_int/cases/basic/ref/out_smoke`
- `make run USE_MERGE=0 REPLAY_DIR=/absolute/path/to/tb_int/cases/basic/ref/out_smoke` forces the direct-path debug mode when you intentionally want to bypass the real merge path.
- `make run QUESTA_HOME=/data1/questaone_sim/questasim`

What it checks:

- replays `lane*_ingress.mem` cycle by cycle,
- drives the SWB wrapper with the real integrated merge path by default,
- compares the observed DMA payload against `expected_dma_words.mem` at the normalized 64-bit hit level, not just the packed 256-bit word level,
- ignores trailing DMA padding after the expected payload count,
- requires `dma_done` and an end-of-event marker.

What it does not need:

- UVM,
- a custom DPI backend.
